## 💼 Highlight Project

### 🚀 Stratum 9 InnerView — AI Hiring Platform

An AI-powered SaaS platform that streamlines recruitment by automating job creation, resume screening, interview workflows, and candidate evaluation.

**Overview**
Built to solve inefficiencies in hiring workflows, the platform enables teams to manage large applicant pools, reduce manual screening, and improve decision-making through structured AI insights.

**Key Capabilities**

* AI-powered job description generation
* Resume screening with skill-based ranking (ATS scoring)
* Automated interview scheduling
* Real-time AI-assisted interview support
* Behavioral analysis during interviews
* Dynamic follow-up question generation
* AI-driven candidate evaluation and scoring
* Transparent feedback dashboards
* Offer letter generation and acceptance tracking
* Role-based access control and audit logs

**Technical Stack**

* Frontend: React.js
* Backend: Node.js
* Database: MySQL
* Infrastructure: AWS EC2

**Impact**

* Reduced manual screening effort significantly
* Improved hiring speed and decision quality
* Enabled structured, unbiased evaluation workflows
